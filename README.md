# IronTempo

IronTempo is a tempo trainer for gym workouts. Control eccentric/concentric speed and pauses to maximize strength, hypertrophy, and form.

## Features
- **Tempo Timer**: Set 4-phase tempos (e.g., 3-1-2-0) with beep + visual cues.
- **Presets**: Hypertrophy (3-1-1-0), Strength (2-0-2-0).
- **Sound Toggle**: Silent mode for public gyms.
- **Simple & Fast**: No login, no tracking.
